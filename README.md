## Homebrew Kong 0.14 Tap

Homebrew tap for [Kong] :beer:

> Kong is a scalable and customizable API Management Layer built on top of
> Nginx.

### Install (stable)

```shell
$ brew tap leandrocp/kong
$ brew install kong
```
